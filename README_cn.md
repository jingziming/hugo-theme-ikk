# Ikk

简约纯粹的中文技术博客风格，基于 Hexo [SimpleBlock](https://github.com/jysperm/hexo-theme-simpleblock) 主题迁移而来

## 预览

[Live](http://jingziming.com)

## Config.toml

```
title = "荆子明's Blog"
theme = "hugo-theme-ikk"

paginate = 10
paginatePath = "page"

[author]
    github = ""
    profile = [
        "* profile line",
        "* profile line"
    ]

[[menu.main]]
    name = "About"
    url = "/about"
[[menu.main]]
    name = "Getting started"
    url = "/getting-started"
```
