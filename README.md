## Config.toml

```
title = "荆子明's Blog"
theme = "hugo-theme-ikk"

paginate = 10
paginatePath = "page"

[author]
    github = ""
    profile = [
        "* profile line",
        "* profile line"
    ]

[[menu.main]]
    name = "About"
    url = "/about"
[[menu.main]]
    name = "Getting started"
    url = "/getting-started"
```
